## Slack message events pipeline code
### App Engine
Make sure you have the gcloud CLI installed, if not download it from [here](https://cloud.google.com/sdk/docs/install)
Deploy by running:
```sh
cd app_negine
gcloud app deploy
```

### BigQuery
Deploy in the BigQuery GCP console
